REAL TIME BUS TRACKER

DESCRIPTION

This project fetch live information from the MBTA (Massachusetts Bay Transportation Authority): https://api-v3.mbta.com, so you can track the LIVE position of two different bus from Line 1 that travels between the MIT campus and Harvard's Campus. You can click on each marker and it will pop up the name of the bus. *Click on the title to redirect to the repository*
